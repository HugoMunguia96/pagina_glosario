Augmented reality:

Is a technology that combines elements of the real world with virtual elements to create an interactive and enriched experience. Through devices such as smartphones, tablets or augmented reality glasses, users can see and experience digital information superimposed on the physical environment.

Web development:

Refers to the process of creating and maintaining websites and web applications. It involves the programming, design, implementation and optimization of websites for operation on the Internet.

Programming language:

It is a tool used to write instructions that a computer can understand and execute. There are different types and versions of programming languages that are used for different purposes and in different contexts.

User interface:

Refers to the way users interact with a software, application or website. It is the means through which users can communicate with the system and perform actions.

Operating system:

It is a software that acts as an intermediary between the hardware of a computer and the programs or applications that run on it. It provides an interface for users to interact with the computer and manage system resources such as memory, processor.

Virtualization:

Is a process by which a virtual or simulated version of a resource or system is created, such as a server, storage, network or operating system. It allows multiple virtual instances to share the physical resources of a computer or server.

Cyberspace:

refers to the virtual or digital environment in which communication, interaction and transmission of information takes place through computer networks, especially through the Internet. It is an abstract space where people can connect, share information.

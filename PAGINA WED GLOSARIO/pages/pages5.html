Modem:

Is a device that allows the connection of a computer or other device to a communication network, such as the Internet. Its main function is to convert digital signals from the computer into analog signals that can be transmitted over telephone lines.

Router:

Is a network device that is used to connect different networks and direct data traffic between them. Its primary function is to send data packets between networks, either within a local network (LAN) or between a LAN and an external network, such as the Internet.

Scanner:

Is a device used to convert physical documents into digital format. It allows you to capture images or text from printed documents and transfer them to a computer or other electronic device for viewing, storage or further processing.

Server:

Is a computer device or program that provides services and resources to other devices or programs, known as clients, on a network. Its main function is to manage and coordinate access to these resources and guarantee efficient communication.

Update:

Used in different contexts to refer to the action of installing a newer version of software, changing the version of an API, modifying existing information or reporting changes in the state or location of something.

HTML:

Is the standard markup language used to create and structure web pages. Provides a set of tags and attributes that define the structure and content of a web page. HTML is the backbone of the World Wide Web.

PHP:

Is a server-side programming language designed primarily for web development. It allows web developers to create dynamic content that interacts with databases. PHP is used to develop dynamic web applications like content management systems.

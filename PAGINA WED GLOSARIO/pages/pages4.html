Firewall:

Is a security measure that acts as a barrier between a computer network and external networks, such as the Internet. Monitors and controls incoming and outgoing network traffic based on predetermined security rules.

API:

Application programming interface. It is an interface that allows interaction with an application or system. An API defines a set of functions, data structures, classes, and objects that programmers can use to build applications on top of it.

Storage:

Refers to the process of saving and preserving objects, data or information in an organized and secure manner for later use or recovery. It can be applied in different contexts, such as the storage of physical products, digital data or information in storage systems.

Python:

Is a popular object-oriented programming language, with a simple and readable syntax. There are various projects and exercises that can be done to learn it, as well as different versions and useful operators.

JavaScript:

It is a widely used programming language that is mainly used for web development. It is a high-level interpreted language that allows developers to add interactivity and dynamic behavior to websites.

Computer virus:

Is a malicious program or code designed to infect and damage computer systems. These programs spread from one computer to another, usually through email attachments, Internet downloads, infected storage devices, or vulnerabilities in the operating system.

Memory card:

It is a portable storage device used to save and transfer data on electronic devices such as digital cameras, mobile phones, tablets and other compatible devices. These cards provide additional storage capacity and are easily interchangeable between different devices.
